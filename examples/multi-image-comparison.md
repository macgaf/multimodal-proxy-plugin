# 示例：多图对比

> 真实运行产物（2026-06-26，doubao-seed-2.0-pro via 火山引擎 Coding Plan）

## 场景

主模型为纯文本模型，用户给出两张纯色拼接图，要求对比颜色布局差异。
主模型无法直接看图，通过 `process_multimodal` 把任务外包给多模态模型。

## 输入

- 图 A：左红右蓝（120×60 PNG，纯色拼接）
- 图 B：左蓝右红（120×60 PNG，纯色拼接）
- 提示词：`["这是图A", "这是图B", "对比这两张图的颜色布局差异"]`

## 工具调用

```
process_multimodal(
  media=["/tmp/mmp_example_a.png", "/tmp/mmp_example_b.png"],
  prompts=["这是图A", "这是图B", "对比这两张图的颜色布局差异"]
)
```

## 真实输出

> 这两张图都是将画面竖直二等分、两色块各占一半的布局，都仅包含红色、蓝色两种颜色，二者布局差异是**左右色块的颜色完全互换**：
> 1. 图A：左侧半区为红色，右侧半区为蓝色，是「红在左、蓝在右」的布局；
> 2. 图B：左侧半区为蓝色，右侧半区为红色，是「蓝在左、红在右」的布局。

## 复现

测试图由 `scripts/test_e2e.py` 动态生成（纯色拼接，无需外部素材）。设置
`ARK_API_KEY` 环境变量后运行：

```bash
.venv/bin/python scripts/test_e2e.py
```
